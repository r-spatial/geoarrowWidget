# Print names and versions of external JavaScript libraries.

Prints names and versions of the external JavaScript libraries used in
`geoarrowWidget`.

## Usage

``` r
extJSLibs()
```

## Details

See e.g. <https://cdn.jsdelivr.net/npm/apache-arrow/package.json> or
<https://cdn.jsdelivr.net/npm/@geoarrow/geoarrow-js/package.json> for
more details on the js depencencies.
